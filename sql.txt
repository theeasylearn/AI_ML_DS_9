How to create table 
create table product 
(
    id int primary key auto_increment, 
    product_code char(8) unique not null, 
    name varchar(32) not null, 
    description varchar(256),  
    category ENUM('Food','Toys','books','perfumes','flowers') not null,
    brand varchar(32) default 'local',
    price decimal(8,2) check (price>0),
    cost_price float check (cost_price>0),
    weight float default 0, 
    stock int default 0, 
    expiry_date date, 
    is_active boolean 
)

Style 1 (Recommended – specify columns )

Aggregate function in mysql 

what is Aggregate function?
function which has numeric field as argument and return single value after processing field is called numeric function 

1. SUM()

Purpose: To calculate the total sum of numeric values in a column.
Use When: You want to add up values (e.g., total sales, total salary, total quantity).

2. MIN()

Purpose: To find the smallest (minimum) value in a column.
Use When: You need the lowest value (e.g., lowest price, earliest date, smallest ID).

3. MAX()

Purpose: To find the largest (maximum) value in a column.
Use When: You need the highest value (e.g., highest salary, latest date, maximum price).

4. AVG()

Purpose: To calculate the average (mean) value of numeric data.
Use When: You want to find the average of values (e.g., average salary, average marks, average price).

5. COUNT(*)

Purpose: To count the total number of rows in the table/result.
Use When: You just want to know how many records exist (including rows with NULL values).
